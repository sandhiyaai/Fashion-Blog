New York Fashion Week Webpage

This repository contains a structured HTML webpage titled “Everyday with Isa”, featuring an article-style layout focused on New York Fashion Week (NYFW). The page includes historical context, styling advice, event guidance, and related fashion content.

Overview:

The webpage presents:

An introduction to the origins of New York Fashion Week

A featured article written by Isabelle Rodriguez

Visual content including multiple images related to fashion events

Tips for attending NYFW, including ticketing, scheduling, and dressing

A related content section

Contact information for the author

The project demonstrates clean HTML structure, use of semantic elements, navigation using anchor links, and balanced text-and-image formatting suitable for a personal blog.

Features:

Header and title section with centered formatting and dividers

Historical background of NYFW sourced from reputable references

Article layout with images embedded throughout

Anchor-based contact link for easy navigation

List of related fashion content for additional exploration

Footer-style contact section with email, phone, and address

File Included:

index.html — Main HTML file containing all content for the NYFW article.

Purpose:

This project is ideal for:

Practicing foundational HTML

Creating static informational web pages

Learning to structure blog-style content

Displaying images, links, and styled text in a clean layout
